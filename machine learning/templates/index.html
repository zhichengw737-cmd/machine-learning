What is Reinforcement Learning?

Help the agent learn by trying actions and earning rewards.

Simple Explanation

Reinforcement learning is learning through rewards and penalties. The agent takes actions and learns which actions are best by trying them.

Agent

The agent is a little computer learner that chooses moves in a grid.

Environment

The grid world is the environment. The agent sees where it is and decides where to go next.

Reward

The agent gets positive reward for good moves and negative reward for wrong moves. It uses reward to improve its choices.

Try the Reinforcement Learning Simulation

Press the button to train the agent and see the path it learns to reach the goal.

Why this helps you learn

  • The agent learns from reward feedback, not direct answers.
  • Positive reward means a good move; negative reward means a wrong move.
  • Over many episodes, the agent finds the best path to the goal.
