Reinforcement Learning Playground

Move the ball to the goal, train the agent, and watch it learn. Click cells to add or remove small obstacles. Use the controls to run training and observe the agent's behavior.

Reward: 0

How to think about Reinforcement Learning

Imagine you are teaching a little robot to find a treasure in a small room made of tiles. The robot does not know the map at first. Each time it moves, you tell it whether that move was good or not: a small penalty for wasting time, and a big reward for finding the treasure. Over many attempts the robot notices which moves tend to lead to the treasure and prefers those.

What students should notice

  • Actions: the robot can move up, down, left, or right. It picks actions and sees what happens.
  • Rewards: a +10 reward for reaching the goal, and -1 for each step. The robot tries to maximize total reward.
  • Exploration vs. practice: sometimes it tries random moves to discover new routes; other times it uses what it already learned.

Parameters

Episodes: think of this as the number of practice runs. More runs give the robot more experience.

Alpha (learning rate): how quickly the robot updates its memory after each example. A small alpha means slow, careful learning; a large alpha means quick but possibly unstable changes.

Gamma (future reward factor): how much the robot cares about future rewards. A higher gamma makes it plan for longer paths to the goal.

Epsilon (exploration chance): the fraction of the time the robot tries something random. A little exploration helps it find better routes it did not see before.

Class activity idea

Ask students to change one parameter at a time and observe the learned path. Which parameter made the robot learn faster? Which made it take longer routes? Discuss why exploration helps in a room with hidden obstacles.

Why this helps you learn

  • The agent learns from reward feedback, not direct answers.
  • Positive reward means a good move; negative reward means a wrong move.
  • Over many episodes, the agent finds the best path to the goal.
