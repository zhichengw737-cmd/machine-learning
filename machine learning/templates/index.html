Why this helps you learn

  • The agent learns from reward feedback, not direct answers.
  • Positive reward means a good move; negative reward means a wrong move.
  • Over many episodes, the agent finds the best path to the goal.

Reinforcement Learning — short

Reinforcement learning is learning from rewards. The agent chooses actions, receives rewards, and updates its policy to get more reward over time.

  • Action: move up/down/left/right.
  • Reward: +10 for reaching the goal, -1 per step.
  • Exploration: try random moves sometimes so the agent discovers better routes.
