📚 Supervised vs. Reinforcement Learning

What is the difference between these two major AI technologies?

📘 Supervised Learning (Features & Labels)

Supervised Learning is like learning with flashcards. We give the computer the exact answer key.

The AI looks at a Feature (the shape of the arrow drawn on the ground) and maps it to a direct Label (the instruction to move Right). Because it is given the answers directly, it learns incredibly fast, but it gets completely stuck if it sees a shape it hasn't been taught yet!

🎯 Reinforcement Learning (Rewards)

Reinforcement Learning is like training a puppy. There is no answer key.

The AI walks around blindly and relies on Rewards (+10 points for a good move, -10 for a bad one). It learns strictly through trial and error over many episodes until it builds a path that gives it the highest possible score.

Core Dimension 🔵 Supervised Learning 🔴 Reinforcement Learning
Core Objective Follows explicit ground-truth labels Maximizes environmental rewards (points)
The "Teacher" Role A human explicitly provides direct correct answers The dynamic environment provides delayed feedback feedback
When an Anomaly Occurs It gets stuck. It cannot resolve states without a known label. Loses points temporarily, explores alternative paths to optimize.
